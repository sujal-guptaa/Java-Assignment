# Java-Assignment

This repository contains my Java programming assignments and practice programs.
The purpose of this repository is to strengthen my understanding of Java programming through regular coding practice.

## 📚 Topics Covered

The assignments in this repository include programs based on the following Java concepts:

* Basic Java Programs
* Conditional Statements (if–else, switch)
* Loops (for, while, do-while)
* Arrays
* Functions / Methods
* Object-Oriented Programming (OOP)
* Bit Manipulation
* **Java Streams API**
* Data Structures Basics

## 🔹 Java Streams Practice

This repository also includes programs that practice the **Java Streams API**, such as:

* Filtering elements using `filter()`
* Mapping elements using `map()`
* Sorting using `sorted()`
* Collecting results using `collect()`
* Finding elements using `findFirst()` and `findAny()`
* Working with `forEach()`

Example concept used:

```java
numbers.stream()
       .filter(n -> n % 2 == 0)
       .forEach(System.out::println);
```
